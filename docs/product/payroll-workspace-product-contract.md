# Payroll Workspace Product Contract

Payroll Workspace UI is future scope.

The backend foundation exposes payroll runs and payroll items for a future operator workspace, but this sprint does not build UI.

Future workspace should expose:

- payroll run queue
- payroll run create/edit
- payroll item management
- worker/source traceability
- gross/reimbursement/deduction/net summaries
- approval lifecycle
- payroll readiness lifecycle
- hold/dispute behavior
- timeline
- audit

Future workspace must not expose payment execution, ACH/card/check payout, payroll provider submission, tax filing, W2/1099 generation, benefits, garnishment, accounting export, bank reconciliation, or worker portal workflows.
